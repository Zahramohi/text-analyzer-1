# Text Analyzer
A simple Python program that analyzes a text.
## Features
- Count the number of words in a text
- Count the number of characters
- Count how many times the word python appears
- Find the longest word in the text
## Example
Input:
```
I love python and learning python programming
```
Output:
```
Number of words: 7
Number of characters: 45
Number of 'python': 2
Longest word: programming
```
## How to run
Run the program with Python:
```
python text_analyzer.py
```
Then enter a sentence and the program will analyze it.
